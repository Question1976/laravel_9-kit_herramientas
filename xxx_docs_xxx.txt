TABLAS Y MODELOS:

xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

Crear tabla 'categorias':
    php artisan make:migration create_categorias_table 
    php artisan make:migration create_categorias_table --create=categorias   (otra forma)
    php artisan migrate

Crear tabla 'productos' con llave foranea hacia 'categorias':
    php artisan make:migration create_productos_table
    php artisan make:migration create_productos_table --create=productos    (otra forma)
    php artisan migrate

Crear tabla 'productos_fotos' (va a tener el ID de la tabla 'productos'):
    php artisan make:migration create_productos_fotos_table
    php artisan make:migration create_productos_fotos_table --create=productos_fotos    (otra forma)
    php artisan migrate

Crear modelo para tabla 'categorias' (un modelo para cada tabla):
    php artisan make:model Categorias

Crear modelo para tabla 'productos' con llave foranea hacia 'categorias' (ver Models/Productos.php):
    php artisan make:model Productos

Crear modelo para tabla 'productos_fotos' con llave foranea hacia 'productos' 
(ver Models/ProductosFotos.php):
    php artisan make:model ProductosFotos
